Dev tool

Sometimes users may need special tools to do unusual things, like recover forgotten master pin code or to fix a corrupted document file. WriterJournal has special dev tools designed just for that. To access these tools, please follow the steps below:

  1. firstly, enable the dev tool in the settings page.
  2. Secondly, navigate to Dev tools page through the navigation menu.


How to recover my forgotten master pin?

To recover your forgotten master pin code, you have to use a special key called recovery key to recover your forgotten master pin, to get a recovery key, you need to send your INSTALL ID to our support team at feedbackpockatapp@protonmail.com, and we will send back the corresponding recovery key to recover your forgotten master pin lock. Please follow the steps below to learn how it works:

  1. Navigate to dev tool page
  2. Click get install ID and copy the result
  3. Send us an email with your Install ID code.
  4. We will verify and send you back your recovery key
  5. Now go to dev page again, click recover master lock pin and paste your recovery key in the text box and click OK, your master pin will be displayed.


How to fix a corrupted document?

Please note that, normally WriterJournal will auto fix a document if it's corrupted with user's confirmation when loading the document, a corrupted document cannot be loaded to the editor page before fixing it, WriterJournal will automatically detect any errors within the document and will try to auto fix it with WriterJournal's diagnose and document health checker components. But sometimes, you may want to use specific methods to fix the corrupted document. If that's the case, then you should go to dev tool page, and click fix a corrupted document button to start the fixing process.

Note that before fixing any document, it's recommended to make a full manual data backup because the fixing process may cause errors in the program.
