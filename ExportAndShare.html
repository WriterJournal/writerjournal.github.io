Export & Share

Export

Writer Journal supports exporting your document to 6 types of file format. Namely, Docx, PDF, HTML, MD, TXT and WjDoc file formats. It also supports exporting your entire folder as a merged document file or as a WjFolder file.

Note that WjDoc and WjFolder file formats are specifically designed for Writer Journal only, which means after exporting to these file formats, you can also import it back.

How to export single document?

Please follow the steps below to export your document. (See the screenshots)

  • Locate your document in the Home page
  • Open Document options menu of that document
  • Click Export and select a file type to export your document
  • Select a path and click Save to start exporting.


How to merge export an entire folder?

Please follow the steps below to export your entire folder containing multiple documents into a single file.(See the screenshots)

  • Locate your folder in the Home page
  • Open Document options menu of that folder
  • Click Export and select a file type to export your folder
  • You will be directed to a Merge export window, from there you can reorder your documents by drag and drop gesture. Note that the default order is the manual order you set for each document, you can set manual order for each document by switching to sort by manual order mode. For more details please visit @link(manual order, manualOrder.html)
  • After reordering, click the export icon on the top panel.
  • Select a path and click Save to start exporting.


How to print your document?

From Android 4.4 (KitKat) onwards, PDF printing directly from Android device becomes a reality, it is a built-in Android OS feature, and all popular web browser apps implemented this feature that helps users directly print any webpage from Android device.

Please follow the steps below to print your document directly from Writer Journal. (See the screenshots)

  • First step is to connect your phone to your printer through local WIFI. (If your printer supports wireless printing, it will establish a WIFI hotspot once you successfully boot it up), connect your phone to the local WIFI of your printer
  • Second, locate one of your documents that you want to print, click Export and choose PDF Export.
  • Third, you will be directed to a PDF preview window, this is an Android OS standard PDF print and export window that has nothing to do with Writer Journal. So each phone may have slightly different window layout, but the basic functions are the same, click View All Printer
  • Fourth, if you successfully connected with your printer, it will be listed in the available devices section, select your printer, and click Print
  • Congratulations and have a good time printing


How to export document or folder to Writer Journal compatible file types and import it back?

Writer Journal introduces self-declared document and folder file types that are completely compatible with Writer Journal app, these are WjDoc for document and WjFolder for folder. You can export your document item to WjDoc file format, and export your folder item to WjFolder file format, and import them back anytime you want.

Please follow the steps below to export your document to WjDoc file format.

  • Locate your document in the Home page
  • Open Document options menu of that document
  • Click Export and select WjDoc export to export your document
  • Select a path and click Save to start exporting.


Please follow the steps below to export your folder to WjFolder file format.

  • Locate your folder in the Home page
  • Open Document options menu of that folder
  • Click Export and select WjFolder export to export your folder
  • Select a path and click Save to start exporting.


After exporting to these Writer Journal compatible file types, you can easily import them back, please visit Import files section for details.

Share

Writer Journal supports 2 types of sharing, namely text sharing and file sharing. You can share your document or folder either as text or as file.

Share single document as text

Please follow the steps below to share your document as text. (See the screenshots)

  • Locate your document in the Home page
  • Open Document options menu of that document
  • Click Share and select Share as text
  • You can either share it as plain text or as Markdown formatted text, select the one you preferred
  • Follow the sharing dialog to finish sharing the text


Share single document as a file

Please follow the steps below to share your document as a file. (See the screenshots)

  • Locate your document in the Home page
  • Open Document options menu of that document
  • Click Share and select Share as file
  • Select a specific file format to share
  • Follow the sharing dialog to finish sharing your document


Share entire folder as merged text

Please follow the steps below to share your entire folder containing multiple documents as a merged text.(See the screenshots)

  • Locate your folder in the Home page
  • Open Document options menu of that folder
  • Click Share and select share as text
  • You can either share it as plain text or as Markdown formatted text, select the one you preferred
  • You will be directed to a Merge share window, from there you can reorder your documents by drag and drop gesture. Note that the default order is the manual order you set for each document, you can set manual order for each document by switching to sort by manual order mode. For more details please visit @link(manual order, manualOrder.html)
  • After reordering, click the share icon on the top panel.
  • Follow the sharing dialog to finish sharing the text.
