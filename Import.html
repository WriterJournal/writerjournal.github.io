Import

WriterJournal currently supports importing 5 different document file formats, namely Plain text TXT file, Microsoft Word DOCX file (beta feature), Markdown text MD file, and WriterJournal self declared WriterJournal Document WJDOC file and WriterJournal Folder WJFOLDER file.

Document Compatibility in WriterJournal

In WriterJournal, all documents are saved in the proprietary WJDOC file format, which is exclusively editable within the WriterJournal app. Therefore, any universal document must be converted to the WJDOC format before it can be edited.

WriterJournal supports seamless conversion of TXT files to WJDOC without any data loss. However, when converting Markdown (MD) and Microsoft Word (DOCX) files to WJDOC, certain formatting details may be omitted, as WriterJournal does not fully support all of their formatting options.

To import a document file into WriterJournal app, please follow the steps below

  1. In the Home Page, click the more option icon on the top right corner and select Import from the drop down menu.
  2. Select an import file type and select the file in the file browser app to start importing.
  3. WriterJournal will show a success window with details regarding the imported document file


Direct Document Import via Share Option in WriterJournal

Another convenient method to import documents into the WriterJournal app is through the share option. Users can share files directly from their file browser app to WriterJournal. If the shared file type is supported by WriterJournal, it will be imported into the currently active folder location.
