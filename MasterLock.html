Master lock

Writer journal introduces a security feature called master lock, it is a 4 digits PIN code that you can use to lock documents and folders. Before using this feature, you have to set up a new master lock PIN code first.

How to setup a master lock?

To setup a new master lock, open Navigation side bar, select Master lock. If you have not yet set up any master lock, you will see a message at the top saying "you must setup a new master pin first", this message indicates that there are currently no master lock available. Enter a four digits PIN code using the number pad, and enter it again for confirmation, once you confirm your master PIN code, you will see a "verified" message indicating your master lock has been successfully configured. And you will be redirected to Home page. Now you have successfully setup a new master lock. After setting up, you can use it to lock your documents and folders. (See the screenshots)


How to lock document or folder?

Before using your master lock, you have to make sure that it is "already verified". Please go to Master lock page, see if it is "already verified" , enter your master pin if it is not verified yet. Once you have it verified. You can start locking your documents. (See the screenshots)


Why my locked documents are still openable?

You may find it confusing that sometimes you found your locked documents are accessible without the need to enter Master PIN, this is not a bug, the reason behind this behavior is that you may have already verified your Master PIN code, thus Writer Journal is not asking you to verify it again until your master lock PIN code validation period expires. You can setup the Master lock validation period in the settings page. (See the screenshots)


After app relaunch: If you select this option, your master lock verified PIN code will expire after you relaunch the app, which means you will have to verify your PIN code again after relaunch.

After 30 seconds: if you select this option, your master lock verified PIN code will expire after 30 seconds, which means you will have to verify your PIN code again after 30 seconds. (Other options work the same way but with longer validation period)

Why my locked documents are not shown in the search page or recent page?

Because you have not yet verified your Master PIN code, you have to verify your Master PIN code first so that your locked documents become searchable and will be visible in search, recent or pinned,colored and tag pages.

You will also not see the locked documents in multiple selection mode or any other pages UNTIL you verify your master PIN code.
